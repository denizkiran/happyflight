# happyflight
Elevating the In-Flight Experience: Introducing HappyFlight

I am thrilled to announce the launch of my latest project: HappyFlight 
Choosing the right seat can make or break a long-haul journey. Is there enough legroom? Is it too close to the galley? Does it actually have a window view? HappyFlight isn't just a seat map; it’s a premium Aviation Decision Support System designed to help passengers navigate these choices with precision.

🌟 Key Features:
Smart Routing & Estimation: Real-time flight duration calculation between major international hubs (JFK, LHR, CDG, etc.).
Interactive Aircraft Visualization: Authentic layouts for a wide range of aircraft, from the Boeing 787 Dreamliner to the Airbus A380, rendered with precision.

Deep Seat Analytics: A proprietary rating system (Excellent to Poor) based on legroom, window alignment, proximity to facilities, and boarding ease.

Premium UX/UI: A sleek, high-contrast dark theme powered by Tailwind CSS, featuring custom "Flying Plane" loaders and fluid state transitions.

🛠️ Technical Architecture:
SPA Architecture: Built as a high-performance Single Page Application for a seamless, "app-like" feel.

Frontend Mastery: Leveraged Tailwind CSS for responsive layouts and Vanilla CSS for complex aircraft fuselage visualizations and keyframe animations.
Logic & State: Robust JavaScript-driven state management handling complex mock data structures to simulate real-world airline configurations.

By merging a passion for aviation with software engineering, I’ve aimed to create a tool that turns a simple list into a visual, data-driven guide.
